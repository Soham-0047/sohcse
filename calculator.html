💡 About the GATE Calculator:
The actual GATE exam provides a virtual scientific calculator (not a physical one). It supports basic arithmetic, trigonometric functions (sin/cos/tan), logarithms (ln/log), square root, power, factorial, and constants (π, e). This calculator mimics that interface so you can practice.

⚠ Note: The GATE calculator does NOT have a graphing feature, memory functions, or programming capability. It's a basic scientific calculator only.
